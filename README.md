# 🚀 KT ADB Tool Pro v2.0.0-PRO-MAX

**KT ADB Tool Pro** là giải pháp quản lý thiết bị Android mạnh mẽ, tinh tế và dễ sử dụng nhất. Được xây dựng trên nền tảng Electron và ADBKit, ứng dụng mang đến trải nghiệm người dùng cao cấp với giao diện Glassmorphism hiện đại.

## ✨ Tính năng nổi bật

### 📱 Quản lý Ứng dụng (App Manager)
- **Phân loại thông minh**: Tự động phân loại Ứng dụng hệ thống, Ứng dụng người dùng và Ứng dụng đã tắt.
- **Dọn rác (Bloatware Removal)**: Tích hợp database rác Xiaomi, Samsung, Google giúp gỡ bỏ ứng dụng thừa chỉ với 1 click.
- **Thao tác hàng loạt**: Gỡ cài đặt, Vô hiệu hóa, Bật lại và Khôi phục hàng chục ứng dụng cùng lúc với giao diện Selection Bar chuyên nghiệp.
- **Cài đặt APK**: Hỗ trợ cài đặt file APK kéo thả với cơ chế shell install an toàn.

### 📂 Quản lý Tệp tin (File Manager Pro)
- **Truy cập Immersive**: Duyệt file với giao diện rộng rãi, hỗ trợ bộ nhớ trong và thẻ nhớ ngoài.
- **Trình xem ảnh thông minh**: Xem trước hình ảnh 4K với chế độ Zoom & Pan mượt mà bằng chuột.
- **Thao tác nhanh**: Hỗ trợ sao chép, di chuyển, xóa và tải tệp tin từ thiết bị về máy tính.

### 🛡️ Bảo mật & Hệ thống
- **Tối ưu hóa**: Các tinh chỉnh hệ thống giúp máy chạy mượt hơn, tiết kiệm pin.
- **An toàn tuyệt đối**: Tích hợp danh sách Blacklist ngăn chặn việc gỡ bỏ các thành phần cốt lõi gây lỗi máy.

### 🎨 Giao diện & UX
- **Glassmorphism Design**: Giao diện trong suốt, hiện đại với các hiệu ứng Blur và Glow.
- **Phản hồi thời gian thực**: Trạng thái thiết bị và ứng dụng luôn được cập nhật liên tục qua ADB.

## 🛠️ Công nghệ sử dụng
- **Frontend**: React, Tailwind CSS, Framer Motion, Lucide Icons.
- **Backend**: Electron, Node.js, ADBKit.
- **State Management**: Zustand.

## 📦 Hướng dẫn Phát triển
1. **Cài đặt dependencies**: `npm install`
2. **Chạy chế độ Dev**: `npm run dev`
3. **Đóng gói ứng dụng**: `npm run build:win`
